# 💻 C# Small Apps

## 📌 Author

Oskar Jasłowski

This is a **personal project** containing a collection of small C# applications built as independent mini-projects.

---

## 🎯 Project Overview

**C# Small Apps** is a set of standalone .NET console applications.

Each project focuses on practicing core C# concepts such as:
- basic calculations  
- object-oriented programming  
- loops and logic  
- simple real-life scenarios  

All applications are organized into separate folders and can be run independently.

---

## 🧩 Included Applications

### 🧮 Kalkulator

Simple calculator performing basic arithmetic operations (addition, subtraction, multiplication, division).

---

### ⚖️ KalkulatorBMI

Application for calculating BMI (Body Mass Index) based on user input.

---

### 📅 LicznikDni

Tool for calculating the number of days between dates or tracking time differences.

---

### 🔁 Petla

Project focused on loops and iteration logic in C# (practice with `for`, `while`, etc.).

---

### 📦 Pozycja Zamówienia

Simple order item system with basic logic like price calculation and quantity handling.

---

### 👨‍💼 ZadanieTestowe

More advanced project using object-oriented programming:
- classes like `Pracownik`, `Kierowca`, `Sprzedawca`  
- inheritance and polymorphism  

---

## 🖥️ How to Run

1. Open the project folder:

```bash
cd C#/ProjectName
```

2. Run the application using .NET CLI:

```bash
dotnet run
```

Or open the `.csproj` file in Visual Studio and run it from there.
